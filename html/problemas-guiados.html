Problemas guiados

Neutralización

Problema guiado: neutralización

Calcula el volumen de HCl 0,10 M necesario para neutralizar 40 mL de Ca(OH)2 0,25 M.

Recuerda:

En una neutralización completa:

moles de H+ = moles de OH-

Paso 1. Calculamos los moles de Ca(OH)2

n = M · V

n = 0,25 · 0,040 = 0,010 mol de Ca(OH)2

Paso 2. Calculamos los moles de OH-

Cada mol de Ca(OH)2 aporta 2 moles de OH-.

moles de OH- = 0,010 · 2 = 0,020 mol

Paso 3. Igualamos con los moles de H+

HCl aporta 1 mol de H+ por cada mol de HCl.

Por tanto, necesitamos 0,020 mol de HCl.

Paso 4. Calculamos el volumen de HCl

V = n / M

V = 0,020 / 0,10 = 0,20 L = 200 mL

Resultado: se necesitan 200 mL de HCl 0,10 M.

Hidrólisis

Problema guiado: hidrólisis de sales

Indica si las siguientes sales darán disoluciones ácidas, básicas o neutras: NaCl, NH4NO3, KCN y CH3COONa.

Recuerda:

Ácido fuerte + base fuerte → sal neutra.

Ácido débil + base fuerte → sal básica.

Ácido fuerte + base débil → sal ácida.

Ácido débil + base débil → depende de Ka y Kb.

NaCl

Procede de HCl, ácido fuerte, y NaOH, base fuerte.

Resultado: disolución neutra.

NH4NO3

Procede de HNO3, ácido fuerte, y NH3, base débil.

El ion NH4+ puede ceder protones al agua.

Resultado: disolución ácida.

KCN

Procede de HCN, ácido débil, y KOH, base fuerte.

El ion CN- puede reaccionar con el agua formando OH-.

Resultado: disolución básica.

CH3COONa

Procede de CH3COOH, ácido débil, y NaOH, base fuerte.

El ion acetato, CH3COO-, puede reaccionar con el agua formando OH-.

Resultado: disolución básica.
